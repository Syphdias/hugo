Bearbeitung von Strings

Es gibt viele Funktionen,um Strings zu bearbeiten,also Teile aus ihnen zu nehmen.
Zum Beispiel habe ich einen Namen,ich will jedoch nur einen Teil von ihm,aus
welchem Grund auch immer.Hier ein Beispiel:
Declare name$,verändert$,verändert%
print "Name eingeben:";
Input name$
verändert$=Left$(name$,3)
print "Die ersten 3 Zeichen des Namens:";verändert$
verändert$=Right$(name$,3)
print "Die letzten 3 Zeichen des Namens:";verändert$
verändert$=Mid$(name$,3,2)
print "Die 2 Zeichen nach dem dritten Zeichen:";verändert$
verändert$=Mid$(name$,1,5)
print "Die 5 Zeichen nach dem ersten:";verändert$
verändert%=Len(name$)
print "Länge von dem angegebenen Name:";verändert%
End
Es dürfte alles Verständlich sein,bis auf Left$($,%),Right$($,%) und Mid$($,%,%).
Hier eine Kurzerklärung der Funktionen:
Left$($,%) In Blatt $ werden von Links % Zeichen gelesen.
Right$($,%) Siehe oben,nur von rechts.
Mid$($,%,%) Aus Blatt $ wird von dem % Zeichen bis zum % Zeichen gelesen.
Len($) Zeigt die Länge von Blatt $ an.
