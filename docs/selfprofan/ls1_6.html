Eingaben des Benutzers...

Bisher konnten immer nur wir über die Blätter/Zahlen bestimmen,jetzt ist es an der Zeit,dass zu
ändern.Der Befehl Input (Deutsch: Eingabe) sorgt dafür,dass der Benutzer auch mal zu Wort kommt:
Declare string1$
print "Hier kann der Benutzer etwas eingeben:"
print ""
Input string1$
print ""
print "Der benutzer hat folgendes eingegeben:"
print ""
print string1$
WaitMouse
End

Dieses programm fordert einen dazu auf,einen beliebigen Text einzugeben.Danach wird dieser wieder
angezeigt.Wenn der Benutzer mit der Maus klickt,wird dass Programm beendet.
Das print "" bedeutet,dass hier eine leere Zeile steht!
