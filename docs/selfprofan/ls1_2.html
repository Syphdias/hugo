Texte


Um Texte oder Blätter(auch Zahlen) wieder auszugeben(anzeigen) kann ich den
Befehl print benutzen.Er gibt einen Text aus.
print "Beliebiger Text"
print blatt1$
print zahl1%
print "Text mit Blatt/Zahl kombiniert:";blatt1$;"und die Zahl";zahl1%;"!"
print "Text mit Blatt/Zahl:",blatt1$

Der auszugebende Text muss immer in Anführungszeichen("") stehen und damit
auch wieder enden.Falls ein Blatt/Zahl angezeigt werden soll,wird ein Semikolons(;) verwendet.
Falls danach noch Text stehen soll,muss noch ein Semikolon(nach dem Blattname) folgen,und danach
(ebenfalls in Anführungszeichen) der Text.
3.Blätter und Zahlen bestimmen
Blätter und Zahlen bestimmen(etwas hineinschreiben).
Angenommen,ich will bei jedem Start des Programmes den Name und die Version ausgeben.
Da weder Name,noch Version feststehen/sich ändern können muss ich das irgendwie anders schaffen,
als mit einem einfachen Print.Dazu benutze ich Print in verbindung mit einem Blatt/Zahl,wie es oben
beschrieben ist.Hier ein kleiner Ausschnitt aus meinem Programm:
Declare name$,version%
name$ = "Mein erstes Programm"
version% = 1
print name$;"Version:";version%
print name$,"Version:",version%
Das hier bewirkt,dass entweder (Beispiel 1) zusammengehängte Wörter
ausgegeben werden,oder (Beispiel 2),durch ein Leerzeichen getrennte Wörter
ausgegeben werden.Ausgabe beider Beispiele:
(1) Mein erstes ProgrammVersion:1
(2) Mein erstes Programm Version: 1
Die Ausgabe lautet:
Mein erstes Programm Version:1
Erläuterung:
name$ = "Wert"
version% = 1
Bei Blättern wird erst der Name des Blattes hingeschrieben,danach ein = und dann der neuen Wert in Anführungszeichen.Diese sind wichtig,da es sonst kein Blatt mehr ist.
Bei Zahlen wird ebenfalls der Name und das = benutzt,aber hier müssen die Anführungszeichen weg!Sonst wird die Zahl nicht als Zahl verstanden,und es gibt einen Fehler.
