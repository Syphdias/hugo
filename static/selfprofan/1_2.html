SelfProfan²:Was sind Compiler,Linker und Runtime?
Ein Compiler macht aus dem Quelltext einen "verschlüsselten" Quelltext,
der dann von der Runtime ausgeführt werden kann.
Dateiendung: *.prf

Was ist eine Runtime?
Die Runtime heißt übersetzt Laufzeit.Sie führt den Quelltext aus.
Schwer vorzustellen,doch wollen wir es mit einer Musik-CD vergleichen:
Du hast dir eine CD(Quelltext) gekauft und willst sie hören.Dazu benötigst
du aber einen CD-Player(Runtime).Nun hast du den CD-Player(Runtime)
gestartet und die CD eingelegt(Quelltext compiliert).Jetzt kannst du die
CD hören(das Programm ausführen).
Dateiendung: *.prc

Der Linker macht aus dem compilierten Quelltext und der Runtime
eine *.exe-Datei,z.B. prog1.exe
Startet man Prog1.exe sieht man das Programm,dass man vorher
programmiert hat.Das geht so:
Die Runtime ist die eigentliche *.exe-Datei.Sie greift auf den compilierten
Quelltext zu,der ja auch in der *.exe-Datei steckt.Nun führt sie ihn aus.
Man kann auch die compilierte Datei direkt mit der Runtime starten,
ohne sie zu linken.
Dateiendung: *.exe
